# Berlitz Operations Platform — Herramientas Recomendadas

## Stack principal

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Hook Form
- Zod
- Supabase
- Supabase Auth
- Supabase Storage
- Supabase Row Level Security
- Vercel
- GitHub
- GitHub Actions
- SheetJS
- pdf-lib
- Docxtemplater
- Playwright
- ChatGPT / Codex
- Cursor o GitHub Copilot

## Uso por área

Next.js + TypeScript será la base de la app web.

Tailwind CSS + shadcn/ui se usará para la interfaz visual.

React Hook Form + Zod se usará para formularios y validaciones.

Supabase se usará para base de datos, autenticación, storage, permisos y APIs.

SheetJS se usará para importar Excel/CSV.

pdf-lib y Docxtemplater se usarán para documentos.

Playwright se usará para pruebas.

Vercel + GitHub se usará para deploy, previews y control de código.
